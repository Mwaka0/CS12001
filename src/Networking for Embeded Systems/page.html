Weather System

Our project is a weather system which includes 2 sensors these are a dust sensor and a temprature sensor, the dust sensor will display the number of particals in the air per cubic metre and the temprature in the surrounding area per degrees celcius, this will be displayed on both the board and the website, the board has 2 lights, red and blue, red is for above room temprature (hot) and blue is for below room temprature (cold), the actual values of these will be displayed on the website.
